# Scrap
This script will scrap the last posts from Ubuntu HandBook site

You need Python3 and bs4 library.

Output will be like this:
<pre>

How to Install f.lux in Ubuntu 16.04, Ubuntu 15.10
 
[hossein@ArchLinux Scripts]$ 
</pre>
